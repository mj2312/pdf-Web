# PDF変換サイト（個人用）
S3とLambdaを利用した画像→PDF変換サイトです。
ユーザーは認証後、Webページ上で画像をアップロードし、ワンクリックでPDFに変換・ダウンロードできます。

## システム構成
```
ユーザー
  ↓
Webページ
  ↓
1. 画像アップロード（jpg/jpeg/png）→ S3（画像用バケット）
2. 変換ボタン押下 → S3にtxtファイルをアップロード（トリガー）
3. Lambda起動 → 画像取得・PDF変換・署名付きURL生成
4. ダウンロードボタン押下 → 署名付きURLから取得
```

## S3バケット構成

| バケット名 | 用途 |
|-----------|------|
| jpg-to-pdf | アップロードした画像を一時保存 |
| marged-pdf | 生成したPDFを保存 |
| presigned---url | 署名付きURL（JSON形式）を保存 |

## 主な機能

### 認証
- IDとパスワードによるログイン必須
- 認証後のみファイル操作が可能

### 処理フロー

**アップロード**
- jpg/jpeg/png形式の画像をS3へアップロード

**変換**
- 変換ボタンでS3にトリガーファイル（txt）をアップロード
- Lambdaが自動起動し、画像をPDFに変換
- 署名付きURLを生成してJSONで保存

**ダウンロード**
- 署名付きURLを取得してPDFをダウンロード

**クリーンアップ**
- ページリロード時に一時ファイル（画像・JSON）を自動削除
- PDFファイルのみ保存

## 使用技術

- **フロントエンド**: HTML / CSS / JavaScript
- **ストレージ**: Amazon S3
- **サーバーレス**: AWS Lambda
- **認証**: ID/パスワードログイン
- **PDF変換**: Python
- **ホスティング**: AWS Amplify
